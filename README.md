# Titanic Exploratory Data Analysis (EDA)

An in-depth Exploratory Data Analysis (EDA) project on the Titanic dataset using Python to uncover patterns, relationships, and factors that influenced passenger survival. My project demonstrates practical data cleaning, statistical analysis, and data visualization techniques commonly used during the early stages of data science and machine learning projects.

---

## Project Overview

Exploratory Data Analysis (EDA) is one of the most important phases of every data science project because it provides a deeper understanding of the data before predictive modelling begins.

In my project, I explored the Titanic dataset to examine passenger demographics, identify missing values, investigate survival patterns, and discover relationships among important variables such as passenger class, gender, age, fare, and embarkation point.

My project combines descriptive statistics with visual analytics to transform raw data into meaningful insights that can guide future machine learning models.

---

## Project Summary

My analysis revealed that passenger survival was strongly associated with demographic and socio-economic characteristics. Female passengers generally experienced significantly higher survival rates than male passengers, while passengers travelling in First Class had considerably better survival outcomes than those in lower classes.

Age distribution analysis showed that the majority of passengers were young adults, with relatively fewer elderly passengers. Missing values were identified and appropriately handled to improve data quality before visualization and analysis.

Correlation analysis indicated only weak linear relationships among most numerical variables, reinforcing the importance of combining statistical summaries with visualization to understand complex datasets.

Overall, my project demonstrates the importance of data cleaning, visualization, and exploratory analysis in extracting meaningful information from raw datasets and establishing a solid foundation for future predictive modelling.

## Key Insights

Some of the major findings from my analysis include:

- Female passengers had a considerably higher survival rate than male passengers.
- Passengers travelling in First Class were more likely to survive than those in Second or Third Class.
- Most passengers were young adults, with relatively few elderly passengers.
- The dataset contained missing values that required appropriate preprocessing before analysis.
- Numerical features exhibited relatively weak linear correlations, highlighting the need for exploratory visualization alongside statistical analysis.
- Exploratory Data Analysis provided valuable insights that can support feature selection and predictive model development.
